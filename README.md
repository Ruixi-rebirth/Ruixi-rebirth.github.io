## Debug Build

```bash
 git clone https://github.com/Ruixi-rebirth/Ruixi-rebirth.github.io.git --branch=blog --depth=1

 cd Ruixi-rebirth.github.io

 cp -r ./static/hugo-theme-stack ./themes/

 hugo server --buildDrafts --forceSyncStatic
```
